# 求职小助 ⚽ AI 实习岗位助手

> 一个为在校大学生打造的 AI 驱动的实习求职助手，打开浏览器就能用。

## ✨ 功能亮点

- 🎯 **精准筛岗**：基于关键词、城市、公司偏好多维度打分
- 🤖 **AI 搜索**：实时调用大模型帮你搜索最新实习岗位
- 📋 **投递管理**：看板式追踪每一次投递进度（已投 / 面试 / Offer / 拒）
- 💪 **面试备战**：AI 出题 + 即时批改 + 自动错题本
- ⚙️ **灵活配置**：关键词权重、城市权重、公司黑白名单，随时调整
- 🔒 **隐私优先**：所有数据存在浏览器本地，不会上传任何服务器

## 🔒 隐私与安全

- **本地存储**：所有配置、岗位数据、投递记录均存储在浏览器 `localStorage` 中
- **API Key 不上传**：你填入的 AI API Key 仅保存在你自己的浏览器，不会被发送到任何第三方服务器
- **直连 AI 服务商**：浏览器直接调用 AI 厂商的官方 API，中间不经过任何代理
- **随时导出备份**：在「求职偏好 → 数据管理」中可以导出/导入 JSON 备份
